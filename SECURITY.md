# Security Policy

## Responsible Use

WDIB is intended for constructive, ethical experimentation and learning.
It must not be used for harmful, abusive, deceptive, invasive, or illegal activities.

## Reporting Misuse or Security Issues

If you discover misuse of WDIB or a security vulnerability:

1. Prefer filing a private security report through GitHub Security Advisories.
2. If private reporting is unavailable, open a GitHub issue with:
   - what happened
   - impact/risk
   - reproduction details (if safe to share)

Please avoid posting sensitive credentials, personal data, or exploit details publicly.
